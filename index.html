What is a Convolution?

Convolution is a method of combining two signals to generate a new output signal. More specifically, we can obtain an output signal \(g(t)\) by convolving an input signal \(x(t)\) and a kernel \(h(t)\).

Note: A kernel can sometimes be referred to as an impulse signal.

Mathematically, the formula for convolution is as follows: $$g(t) = x(t) * h(t)$$

Convolving a signal can have many useful applications, such as smoothing signals, filtering, and edge detection.

How do you perform a Convolution?

You can think of it as a small window, the kernel in this case, sliding across the input signal. At each timestep of the sliding window, we perform pairwise multiplication on the overlapping values and sum them to produce a new output value.

The process can be better understood through the visualisation below, which takes you through the step-by-step process to convolve a randomly generated noisy sinusoidal input signal using a simple kernel \([1, 2, 3]\).

Note: The values of the kernel are distinct to facilitate the tracing of computation.

Notice that the kernel is "flipped" when sliding across the input signal. This comes directly from the definition of the convolution integral: $$(x * h)(t) = \int_{-\infty}^{\infty} x(\tau)\, h(t - \tau)\, d\tau$$

The term \(h(t - \tau)\) evaluates the kernel in reversed time, which is equivalent to flipping the kernel before shifting it across the input signal.

In this demonstration, the effect on the output may appear subtle because the chosen kernel is relatively simple. We will explore kernels that are more expressive in later sections, where the effects of convolution become more visually apparent.
